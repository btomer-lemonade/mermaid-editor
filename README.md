# Mermaid Editor

A Mermaid diagram editor with powerful built-in commands.

## Features

### Smart Code Actions

Context-aware editor actions that intelligently insert code based on your cursor position and selection, such as inserting a new node, swapping the arrow direction, and auto-formatting the code.

Press `?` to view the full list of actions and keyboard shortcuts.

### Powerful Editor

Full-featured code editor with custom Mermaid language support:

- Syntax highlighting for all Mermaid keywords and arrow types
- Auto-completion
- Bracket matching and auto-closing pairs
- Standard editor features like duplicate line, select all occurrences, and multi-cursor editing
